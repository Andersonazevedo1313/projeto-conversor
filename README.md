💱 Conversor de Real para Dólar
Projeto simples de conversão de moeda que transforma valores de Real (BRL) para Dólar (USD) utilizando HTML, CSS e JavaScript.
Este projeto foi desenvolvido para praticar conceitos básicos de desenvolvimento web, como manipulação do DOM, captura de valores em inputs e cálculos com JavaScript.
🚀 Funcionalidades
✔️ Inserir valor em Real (R$)
✔️ Converter para Dólar (US$)
✔️ Exibir o resultado instantaneamente
✔️ Interface simples e fácil de usar
🛠️ Tecnologias utilizadas
HTML5 → Estrutura da página
CSS3 → Estilização da interface
JavaScript → Lógica de conversão
