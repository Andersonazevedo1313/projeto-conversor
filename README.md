# 💸 Conversor de Moedas: Real para Dólar

Um projeto prático e responsivo desenvolvido para converter valores de Real (BRL) para Dólar (USD) em tempo real, utilizando taxas de câmbio definidas no script.



## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as tecnologias fundamentais do desenvolvimento web:

* **HTML5**: Estruturação dos elementos da página e formulários.
* **CSS3**: Estilização moderna, utilizando gradientes e layouts responsivos.
* **JavaScript**: Lógica de conversão, manipulação do DOM e formatação de moedas.

## 🛠️ Funcionalidades

- [x] Entrada de valor em Reais.
- [x] Conversão automática (ou via botão).
- [x] Formatação de moeda internacional (ex: R$ 5,00 e $ 1.00).
- [x] Design adaptável para dispositivos móveis.
